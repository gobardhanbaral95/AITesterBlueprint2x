# AI Tester Blueprint 2x

Welcome to the **AI Tester Blueprint 2x** repository! This project serves as a comprehensive guide to understanding and integrating Artificial Intelligence (AI) and Large Language Models (LLMs) into modern software testing and Quality Assurance (QA) workflows.

The repository is structured into different chapters and projects, documenting our step-by-step journey:

## 📖 Chapter 1: LLM Basics & Local Test Generation
**Folders:**
- `Project_01_LocalLLMTestGenerator`
- `Project_01_LocalLLMTestGenerator_Antigravity`
- `LocalLLMTestGenBuddy`

**Overview:**
In this foundational chapter, we explored the basics of Large Language Models (LLMs). We focused on how to leverage local LLMs for generating deterministic, self-healing test automation scripts and establishing strict rules (like the Anti-Hallucination rules and B.L.A.S.T. master system prompt) to prevent AI drift or fabricated outputs.

## 📖 Chapter 2: Prompt Engineering & Frameworks
**Folder:**
- `Project_02_RICE_POT_Selenium_FW`

**Overview:**
Here, we discussed the power of **Prompt Engineering** in automation. We introduced the **RICE-POT** prompt framework (Role, Instructions, Context, Example, Parameters, Output, Tone) and used it to generate an enterprise-grade Selenium WebDriver automation framework in Java, integrating TestNG and Page Object Model patterns—ensuring zero bad coding practices and strict compliance with production-level standards.

## 📖 Chapter 3: Advanced Prompt Engineering with Playwright
**Folder:**
- `Project_03_RICE_POT_Playwright_Advance_FQ`

**Overview:**
Building upon Chapter 2, this chapter dives into advanced prompt engineering techniques applied to modern web testing tools—specifically using the **RICE-POT** framework to architect and build robust end-to-end testing solutions with **Playwright**.
